Scope and Purpose

Definitely Not Craigslist is a growing share economy website. Our goal is to be as inclusive as possible. We want people of all ages and with all sorts of technology capabilities to use our services. We do this by keeping the site simple for everyone to use. Our style guide is designed to inspire developers that also want to make highly accessible sites and for our developers to keep building on this idea. The layout is responsive and will look great on many different screens. To achieve this, Bootstrap v4.3.1 was used as our framework. Much of the styles and layout is based on Bootstrap's structure and functionality. We used Marvel and Bootstrapâ€™s Style guide as inspiration for ours.

Responsive Grid

Description

Definitely Not Craigslist uses Bootstrap's 12-column grid that allows for a wide variety of layouts. The columns are placed within the .row class, which is in a .container class. The .container class defines the container of elements. It changes its width based on the viewport. If the entire width of the viewport is preferred, the .container-fluid class can be used instead. Columns will automatically size themselves to fill in the container. By default, rows have flex wrap and will wrap content based on the size of the viewport. The .flex-nowrap class can be used to disable this style.

Equal Width Columns

The simplest way to create columns is by using .col. This method will create columns with equal width. The sizes will stay the same as the viewport changes.

Specific Width Columns

If specific widths are desired for elements, .col can be appended with a dash and a number between between one and twelve. The number determines the width of the column relative to the row.

Dimensions

Description

Definitely Not Craigslist uses Bootstrapâ€™s .container class as the main container. This gives us a responsive box in the center of the page that changes size based on breakpoints. Bootstrap has default margins and padding set for different elements. Their spacing classes can be used if you want something other than these defaults. These classes are easy to use because how they're named. They assign a rem size to the applied margin or padding. Bootstrap's default margins and padding are generally appropriate and should be left as is.

Guide

Bootstrap offers an easy way to add margins and padding. There are simple classes to change margins and padding without having to code any CSS. A complete guide to Bootstrap's spacing can be found here.

Usage Suggestions
  • <h1> headings should be given the .my-3 class for enough spacing from the nav bar and main content.
  • Labels should have mt-4 for proper spacing between filters.
  • Cards should be given my-3 so they have sufficient spacing between them and pb-4 to increase their size.
  • Multiple containers should have the .my-3 class for adequate gaps between them.
  • Multiple rows should have the .mb-2 class for enough space between them.

Description

To provide maximum readability across multiple devices, Bootstrap uses a font stack. The font will change depending on the device being used. Definitely Not Craigslist does not make any changes to this font stack. The default font and browser combinations are:

Description

Throughout the website, with the exception of the nav bar items, hyperlinks have a font size of 1 rem and a color of #0056b3 while being hovered (a dark shade of blue) and a color of #007bff (a lighter shade of blue) without being hovered. These are the default colors that Bootstrap uses. They are similar to unstyled links and should easily be recognized by users. All nav bar links are bold with the brand link at 30px. Nav bar links to other pages are gray and turn black when hovered. The current page link is always black. There is no text decoration for any link.

Usage Suggestions

Input

Input fields can be found on various Definitely Not Craigslist pages such as the Create a Listing or in Profiles. The .form-group class should be applied to parent containers of inputs to provide adequate padding. Typical form elements and their usage can be found in this section.

Select

Select menus can be found on the search bar and the Create a Listing pages. Bootstrapâ€™s .form-control class is applied to style the select element.

Text Input

Standard text input used to get information such as email or name from users. Bootstrapâ€™s .form-control class is applied to style the select element.

Slider

Sliders can be found under search filters. Input tags must have their types set to range to get this function. They allow the user specify a specific distance. They typically have a range of 1 to 50 miles, but can be changed to increase search radius as needed.

Checkboxes

Checkboxes are found under search filters. Input tags must have their types set to checkbox to get this function. They should be used to allow users to filter out results by availability. The .mx-auto class should be applied to center the checkboxes in their container.
